# yolo-format-labeling-tools
Self use labeling tool, only supports YOLO format

# How to use
1) Modify 'DATASET-DIR' in app.cy to the root path of your dataset
2) Modify the "LABEL'ORDER" in app.Py to the names of various categories in your dataset
3) run ```python app.py```
4) Open in browser http://localhost:5000, the port number can be modified by oneself
